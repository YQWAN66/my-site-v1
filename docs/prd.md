# 个人主页需求文档

## 1. 应用概述

### 1.1 应用名称
二川的个人主页

### 1.2 应用描述
这是一个个人展示型网站，用于展示二川的个人简介、工作经历及专业能力，并提供与数字分身互动的聊天功能，方便访客快速了解二川并获取相关信息。

### 1.3 目标访客
- 小红书粉丝
- 朋友
- 面试官

## 2. 核心功能

### 2.1 个人信息展示区

#### 2.1.1 头像
- 支持本地上传修改头像

#### 2.1.2 基本信息
- 显示名字：二川
- 显示一句话介绍：建筑学硕士转行中厂担任AI产品经理
- 支持在线修改名字和一句话介绍

#### 2.1.3 详细个人信息
- 现在主要在做：整理自己的作品：营销、客服AI语音机器人；分享转行学习经验分享；闲余时间做AI提效工具
- 兴趣：探索市场AI动态、使用AI做出更多好玩有用的工具
- 特色特点：喜欢把复杂问题讲成人话
- 支持在线修改个人信息内容

### 2.2 数字分身聊天区

#### 2.2.1 聊天功能
- 访客可通过输入框向数字分身提问
- 数字分身根据预设知识库回答问题

#### 2.2.2 预设知识库
- 职业身份：AI产品经理
- 最近在做的事：搭个人主页网站、探索AI短剧、养龙虾openclaw
- 擅长/关心的方向：AI能够代替人做什么
- 常见问题及答案：
  - 你做了哪些成功的AI应用？（回答：营销、客服AI语音机器人等作品）
  - 怎么联系你？（回答：提供社媒联系方式）
  - 你的社媒有哪些？（回答：提供社媒账号信息）

## 3. 技术说明

### 3.1 响应式设计
- 需要适配手机端
- 需要适配平板端
- 需要适配桌面端

### 3.2 性能要求
- 页面加载速度快
- 交互响应及时
- 支持流畅的聊天交互体验

## 4. 设计要求

### 4.1 视觉风格
- 简约
- 清爽
- 稍微有一点科技感
- 太平花哨
- 要有动感

### 4.2 配色方案
- 主色调：深蓝色和白色

### 4.3 交互体验
- 信息层次清晰
- 页面结构清楚
- 功能易用性强

## 5. 其他说明

### 5.1 内容管理
- 头像、名字、一句话介绍、个人信息等支持在线修改
- 数字分身知识库支持更新维护

### 5.2 发布要求
- 生成可预览的原型版本
- 优先保证结构清晰和功能可用
- 避免过度复杂的设计
